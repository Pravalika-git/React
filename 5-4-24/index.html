JavaScript Spread and Rest Operators

The spread (...) and rest (...) operators in JavaScript look identical but serve different purposes depending on where and how they're used.

Spread Operator (...)

The spread operator unpacks or expands elements from an array, object, or string.

Arrays

const arr1 = [1, 2, 3]; 
const arr2 = [4, 5, 6]; 
const combined = [...arr1, ...arr2];
Output (combined): 
const original1 = [1, 2, 3]; 
const copy = [...original1];
Output (copy): 
const numbers = [1, 2, 3]; 
console.log(Math.max(...numbers));

Objects

const obj1 = { a: 1, b: 2 }; 
const obj2 = { c: 3, d: 4 }; 
const merged = { ...obj1, ...obj2 };

Rest Operator (...)

The rest operator collects multiple elements into a single array.

Function Parameters

function sum(first, ...others) {
  return others.reduce((total, num) => total + num, first);
}
console.log(sum(1, 2, 3, 4));

Destructuring Arrays

const [first, second, ...rest] = [1, 2, 3, 4, 5];
